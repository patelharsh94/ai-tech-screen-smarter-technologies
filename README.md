## Features
- Chainlit's beautiful UI for communicating with the Thoughtful AI GPT
- Can ask any default questions and will be able to get a good answer.
- Can ask malicious questions and those questions will be replied to in a safe and respectful way.
- Can ask question that the GPT doesn't know about and it will answer in a respectful way that it doesn't know.
- If the user asks irrelevant question, the GPT will steer the conversation back to Thoughtful AI, and it's services.
